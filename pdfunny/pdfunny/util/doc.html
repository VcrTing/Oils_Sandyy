妳好，歡迎使用 PDF 平臺

這個文章裏，包含所有妳想要知道的東西？還請繼續看下去......

想要使用這個平臺導出 PDF，就需要做兩步驟： 第壹步就是傳送 HTML 內容過來，我們可以把內容保存為 HTML 文件； 第二步，把我們發給妳的文件名發送給我們，我們可以通過文件名，查找到妳的 HTML 文件，然後將文件轉換成 PDF 再送回給妳 PDF 的下載鏈接。

發送妳的 HTML 內容給我們

需要使用 POST 方法，發送 "FORM BODY" 給我們......

請發送到 /pdf/html/save

我們收到含有 html 名稱的參數後，會將參數的內容儲存到 HTML 文件中

之後，我們會返回壹些值給妳：


{ named: "default_20220513172125_001" }

憑借這個 named 值，可以拿到 PDF 哦!!!

請繼續往下看......


妳拿到的 NAMED 很有用

使用 GET 方法，發送妳的 named 值給我們......

請發送到 /pdf/html/convert/[named]

請註意 [named] 這裏填入妳的 named 值!!!

這樣，我們會將 PDF 文件的下載鏈接返回給妳:


{ pdf: "https://.../pdf/default_20220513172125_001.pdf" }

這樣應該就懂了 吧!!!
